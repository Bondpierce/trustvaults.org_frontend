How does trustvaults operate?

In basic terms, Trustvaults (also know as autopilot trade ) work by retrieving data from an exchange. Using technical indicators and its pre-defined algorithm, the robot calculates a trade, and sends the execution details to the API. Finally, the same interface sends the order of trade, which is then automatically executed by the users exchange account.

They can also be used to carry out trades based on your rules by inputting pre-defined conditions. This is particularly useful for crypto markets because they are active 24/7 all year round.
